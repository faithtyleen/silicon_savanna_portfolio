Silicon Savanna Portfolio – Week 2
Project Overview
Vanilla PHP portfolio built without frameworks for the DevOps internship Week 2: “From Student to Practitioner.”

Requirements Met
Day 1: Vanilla HTML/CSS from Figma design
Day 2: Manual router with clean URLs
Day 3: Contact form with PHP validation
Day 4: Apache web server configured
Day 5: Documentation and reflection submitted

Technologies
Backend: Vanilla PHP
Frontend: HTML5, CSS3, JavaScript
Server: Apache with .htaccess
No frameworks used (React, Angular, Bootstrap, Tailwind)
Routing:Manual PHP router (`index.php`)
Styling: Custom CSS (no Bootstrap/Tailwind)

Quick Start
Using XAMPP
Place folder in C:\xampp\htdocs\silicon-savanna-portfolio\
Start Apache in XAMPP Control Panel

Visit: http://localhost/silicon-savanna-portfolio/
